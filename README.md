# Smart Attendance Tracking System Using QR Codes

## Description
This project is a full-stack web-based Smart Attendance System designed for classrooms. It automates attendance using QR codes, reducing errors and saving time compared to manual attendance.

Students scan a unique QR code for each class session, and the backend validates and stores attendance. Instructors can generate QR codes and view live attendance.

---

## Features
- Instructor dashboard to generate QR codes
- Student QR scanning for attendance
- Attendance validation (no duplicate check-ins)
- Live attendance dashboard for instructors
- Real-time updates
- Simple and clean frontend design

---

## Technology Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **Database (future version):** MySQL  
- **QR Code Generation:** qrcode library  
- **QR Scanning:** html5-qrcode  
- **Version Control:** Git & GitHub
